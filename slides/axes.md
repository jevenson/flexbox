# Main Axis & Cross Axis

main axis runs from left to right <!-- .element: class="fragment" -->

cross axis runs from top to bottom <!-- .element: class="fragment" -->

children flow from start to end of main axis <!-- .element: class="fragment" -->

by default! <!-- .element: class="fragment" -->

Note:
- 1st concept
- Main Axis: default is the horizontal plane from left to right
- Cross Axis: default is the vertical plane from top to bottom
- These properties are default, and can be changed
